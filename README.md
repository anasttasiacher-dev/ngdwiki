# NGDWiki

База знаний по нарративному гейм-дизайну — **https://ngdwiki.ru**

Сайт на **MkDocs Material**. Контент — обычные Markdown-файлы в папке `docs/`,
редактируется в **Obsidian**, публикуется автоматически.

## Как это работает

```
Правишь .md в Obsidian  →  изменения уходят на GitHub  →  сайт автоматически пересобирается и обновляется
```

## Структура

| Путь | Что это |
|---|---|
| `docs/` | контент сайта (статьи + картинки) |
| `mkdocs.yml` | настройки сайта: название, тема, меню |
| `overrides/` | доработки темы (в т.ч. счётчик Яндекс.Метрики) |
| `hosting/htaccess` | редиректы для хостинга |
| `.github/workflows/` | автосборка и публикация |

## Как вести сайт

Контент правится в **Obsidian** (папка `docs/`); изменения автоматически публикуются на сайт.
Инструкция по настройке — у владельца.
